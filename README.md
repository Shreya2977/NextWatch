# NextWatch: Movie Recommender System 🎬

A content-based movie recommender system that suggests the top 5 most similar movies based on a user's query. This project processes raw movie metadata, applies text vectorization techniques, and serves the recommendations through an interactive, clean **Streamlit** web application.

---

## 🚀 Features
* **Content-Based Filtering:** Recommends movies by analyzing metadata likeness (genres, keywords, cast, and director).
* **Top-5 Recommendations:** Instantly computes and displays the 5 closest matches for any selected movie.
* **Interactive UI:** Built entirely with Streamlit for a smooth, responsive user experience.
* **Optimized Pipeline:** Features clean data parsing, text preprocessing, and saved structural matrices for instant inference.

---

## 🛠️ Tech Stack & Concepts
* **Web Framework:** Streamlit
* **Data Manipulation:** Pandas, NumPy
* **Machine Learning & NLP:** Scikit-learn (`CountVectorizer`), Cosine Similarity
* **Data Serialization:** Pickle
* **Dataset Source:** Kaggle (TMDB 5000 Movie Dataset)

---

## 🧠 Core Architecture & Workflow

The system is split into two primary phases: data modeling (Jupyter Notebook) and deployment (Streamlit).

### 1. Data Preprocessing & Feature Engineering
* Extracted structural metadata from the raw Kaggle dataset, merging multiple sources on the movie ID.
* Handled missing attributes and cleaned JSON-string columns (like genres, cast, and crew) into clean Python lists.
* Extracted the director's name from the crew metadata and isolated the top 3 actors from the cast.
* Combined the movie overview, genres, keywords, cast, and crew into a single comprehensive `tags` text block.
* Cleaned the text by converting it to lowercase and applying stemming to normalize words (e.g., "loving", "love", and "loved" become "love").

### 2. Vectorization & Similarity Matching
* Converted the textual `tags` into a 5,000-dimensional numerical space using **Bag-of-Words** vectorization (`CountVectorizer`), isolating the top 5,000 most frequent terms while excluding standard English stop words.
* Calculated the **Cosine Similarity** matrix across all vectors to measure the angle of similarity between movies in the high-dimensional space.
* Exported the processed movie data dictionary and the computed similarity matrix as `.pkl` files using the `pickle` library.

### 3. Web Application Deployment
* Built a lightweight application in `app.py` that loads the serialized files on startup.
* Created a searchable dropdown selector using Streamlit's UI components.
* Developed an internal selection function that identifies the index of the user's chosen movie, references the similarity matrix vector, sorts the top matches, and renders the top 5 structural results.
